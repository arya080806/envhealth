## 设计方法记录

- 移动端优先：默认宽度约 430px，横屏桌面时扩展到 tablet 宽度。
- 深色沉浸：暗森林背景叠加深绿色纵向渐变遮罩。
- 玻璃面板：底部半透明深色面板承载 `Start a session` 与开始按钮。
- 高对比标题：左上大字号白色中文标题，英文副标题采用大写字距。
- 视觉纹理：使用细纵向网格和浅色高光线条增强沉浸感。
