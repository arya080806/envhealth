# 新手指引内容计划

## 目标

为用户提供一个可主动打开的新手指引入口，帮助理解从上传环境照片到生成方案、查看记录的完整流程。指引不在首次进入时自动弹出，避免打断首页体验。

## 入口位置

- 入口名称：新手指引
- 推荐图标：问号或指南图标
- 位置：页面右上角区域
- 规则：与用户按钮、保存按钮、返回按钮保持间距；如果页面右上角已有用户按钮，则新手指引放在用户按钮左侧；如果页面右上角已有保存按钮，则放在保存按钮左侧或标题栏右侧第二位。
- 展现方式：点击后打开底部抽屉或居中弹窗，用户可关闭、下一步、上一步。

## 指引结构

### 1. 首页

标题：从这里开始

说明：点击“开始体验”进入照片选择页。这个应用会围绕一张真实环境照片，帮助你尝试更舒适、更有恢复感的空间变化。

重点提示：

- “开始体验”进入上传或模板选择。
- 右上角用户按钮进入账号信息。
- 新手指引入口可随时再次打开。

### 2. 拍摄环境页

标题：选择一张环境照片

说明：你可以拍摄、上传照片，也可以直接选择系统提供的环境场景。照片会作为后续改造的基础。

重点提示：

- 上传真实场景适合改造自己的空间。
- 系统模板适合快速体验完整流程。
- 下方场景卡片可直接点击进入下一步。

### 3. 选择改造模式页

标题：选择参与方式

说明：这里展示当前选择的环境图像，并提供不同参与深度的改造方式。

重点提示：

- 自由创作：直接摆放树木、座椅、水景等元素。
- 灵感创想：用手绘或草图表达想法。
- 对话改造：用文字描述你希望空间带来的感受。
- 智能参数：通过参数快速生成推荐方案。

### 4. 自由创作页

标题：把元素放进环境里

说明：点击或拖拽下方元素到画布中，选中后可以缩放、旋转、删除、复制或置顶。

重点提示：

- 先选择元素分类，再点击具体元素。
- 元素放好后点击“AI 融合生成”。
- 画布中的图片会铺满编辑区域，生成时以当前布局为参考。

### 5. 灵感创想页

标题：画出你的想法

说明：用简单线条或涂抹表达想加入的空间感觉，系统会结合原图理解你的创作意图。

重点提示：

- 不需要画得精确，表达大概区域和方向即可。
- 可以用不同工具表达植物、水景、路径或氛围。
- 完成后提交生成空间方案。

### 6. 对话改造页

标题：用语言描述期待

说明：你可以告诉系统希望空间更安静、更开放、更自然，或描述一个具体场景。

重点提示：

- 描述感受比描述技术细节更重要。
- 可以补充“不想要”的元素。
- 系统会把文字转化为改造方向。

### 7. 智能参数页

标题：用参数快速调整

说明：通过滑杆调整绿化程度、人造元素、光线氛围和空间活力，适合想快速得到结果的用户。

重点提示：

- 参数越高不一定越好，适度变化更容易带来舒适感。
- 可以多次尝试不同组合。
- 提交后生成对应方案。

### 8. 结果页

标题：查看生成方案

说明：结果页展示生成后的环境图像，并提供保存、继续调整或记录评估的入口。

重点提示：

- 不满意可以返回上一步继续调整。
- 满意后保存到草稿箱或记录。
- 建议记录当前方案带来的感受。

### 9. 草稿箱和记录页

标题：回看你的空间变化

说明：草稿箱保存未完成方案，记录页用于查看历史生成和评估结果。

重点提示：

- 草稿适合继续编辑。
- 记录适合比较不同方案。
- 可作为后续研究或个人复盘依据。

## 文案风格

- 简短、安静、清楚。
- 避免技术术语。
- 每一步控制在 1 个标题、1 段说明、2 到 3 条重点提示。
- 不使用强制教学语气，强调“你可以”“适合”“建议”。

## 交互建议

- 默认不自动弹出。
- 点击右上角入口后打开。
- 弹窗中显示当前页相关指引，并提供“查看完整流程”。
- 完整流程可以用分页卡片展示。
- 用户关闭后不写入强制已读状态，入口始终可用。

## 后续实现清单

1. 新建通用 `GuideButton` 或 `guide_entry` 组件。
2. 新建 `GuideDialog` 或 `GuideDrawer` 组件。
3. 为首页、上传页、模式选择页、四个改造页、结果页、记录页配置对应指引内容。
4. 在每个页面顶部右侧插入入口，并按页面已有按钮动态调整位置。
5. 添加移动端样式检查，确保不与用户按钮、保存按钮、返回按钮重叠。
6. 用浏览器验证首页、模式页、自由创作页、结果页的入口位置和弹窗可读性。
